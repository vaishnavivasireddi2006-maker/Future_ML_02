# 🎫 Support Ticket Classification System

This project is a Machine Learning-based system that automatically classifies customer support tickets into different categories such as Billing, Technical Issues, Account Issues, and General Queries.

It helps businesses improve customer support efficiency by reducing manual effort and prioritizing issues faster.

---

## 🚀 Features

- Automatic ticket classification using NLP
- Text preprocessing (cleaning, stopword removal)
- Feature extraction using TF-IDF
- Machine learning model using Scikit-learn
- Interactive web app built with Streamlit

---

## 🧠 Tech Stack

- Python
- Scikit-learn
- NLTK (NLP)
- Streamlit

---

## ⚙️ How It Works

1. User enters a support ticket description
2. Text is cleaned and processed
3. Converted into numerical features using TF-IDF
4. ML model predicts the ticket category
5. Result is displayed in the web app
